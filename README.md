## üöÄ Getting Started

### Prerequisites
To get started, you will need the following:
1. **Databricks account**: Sign up for [Databricks Community Edition](https://community.cloud.databricks.com) if you don‚Äôt have one.
2. **Python Installed**: Preferably Python 3.8 or newer. You can download it [here](https://www.python.org/downloads/).
3. **An IDE or Text Editor**: Such as VS Code, PyCharm, or Jupyter Notebook.

### Installation

Follow these steps to set up the project on your local machine:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/aviral-bhardwaj/databricksLearning.git
   cd databricksLearning
   ```

2. **Set up the environment**:
   Install the necessary libraries using pip:
   ```bash
   pip install -r requirements.txt
   ```

3. **Databricks Configuration** (Optional but recommended):
   - Set up your Databricks workspace as explained in [Databricks documentation](https://docs.databricks.com/getting-started/index.html).
   - Upload the Python scripts or notebooks from this repository to your workspace.

4. **Run examples**:
   - Execute the provided Python scripts or notebooks.
   - Customize the files to suit your learning path and integration with Databricks.
